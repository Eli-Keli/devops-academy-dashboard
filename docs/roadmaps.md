# Learning Roadmaps Documentation

The DevOps Academy curriculum covers 9 progressive tracks, preparing Eli for his Safaricom internship.

---

## 1. Track Pipelines

1. **Linux Fundamentals**: CLI navigation, Systemd, logs, cron, processes.
2. **Networking Fundamentals**: IP routing, DNS resolution, UFW firewalls, curl.
3. **Git & GitHub Workflows**: Branching strategies, rebase, SSH keys.
4. **Docker & Containers**: Layer caching, Compose, container bridges.
5. **CI/CD Pipelines**: Automated test workflows, actions.
6. **Cloud Fundamentals**: Compute instances, IAM scopes, secure VPCs.
7. **Infrastructure as Code**: Declarative Terraform resource configurations.
8. **Monitoring & Observability**: Prometheus metrics queries, Grafana boards.
9. **Kubernetes & Orchestration**: Deployment clusters, ingress, services.

---

## 2. Advancement Rules
* Tracks must be completed sequentially.
* Upcoming tracks remain locked until the active track progress reaches 100% and receives an "advance" critique verdict from the Lab Instructor.
