# DevOps Academy Dashboard

A specialized, read-focused learning command center, progress tracker, and roadmap visualizer designed to support Eli's DevOps learning journey in preparation for the Safaricom internship in June 2026.

---

## 1. Project Purpose
The dashboard is designed to answer one central daily question: **"What is my operational focus today, what is my progress, and what actions must I take next?"** 
Rather than being a heavy LMS or interactive workspace, it serves as a high-clarity **visual read model** summarizing state generated by AI learning agents (Hermes/Cortana) and stored in local files.

---

## 2. Architecture Overview
This application is built on top of a modern full-stack TypeScript environment:

```
┌────────────────────────────────────────────────────────┐
│               DevOps Academy Dashboard                 │
├──────────────┬──────────────┬───────────┬──────────────┤
│  Dashboard   │   Roadmap    │   Labs    │   Reviews    │
│  (Missions)  │  (Progress)  │(Practice) │ (Assessments)│
└──────────────┴──────────────┴───────────┴──────────────┘
```

* **Frontend Library**: React 19 (`react`, `react-dom` `^19.2.0`).
* **Full-Stack Framework**: [TanStack Start](https://tanstack.com/router/v1/docs/framework/react/start/overview) which utilizes the **Nitro** server engine under the hood.
* **Routing**: [TanStack Router](https://tanstack.com/router) with strict, compile-time file-based routing.
* **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) using CSS custom property mappings, modern OKLCH color palettes, and glassmorphism cards.
* **Icons**: [Lucide React](https://lucide-react.dev/).
* **Package Manager**: NPM / Node `v24.15.0`.

## 4. Development Workflow

### Requirements
* **NodeJS** (`>=v20.x`)
* **NPM** (`>=v10.x`)

### Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local development server:
   ```bash
   npm run dev
   ```
3. Open your browser and navigate to `http://localhost:3000`.

### Production Build compilation
Verify that the project compiles cleanly (both Client and Server environments):
```bash
npm run build
```
This performs static bundle generation, TypeScript compiles, and asset optimizations.
